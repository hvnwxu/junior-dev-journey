### 예시)

#### 1. 문제 이해
- 입력: 점수 1개
- 출력: 점수에 맞는 등급

#### 2. 풀이 계획
1) 점수를 입력 받는다  
2) 점수가 90 이상이면 A 출력  
3) 80 이상이면 B 출력  
4) 나머지는 조건에 따라 출력한다

#### 3. 코드
```python
score = int(input())

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
elif score >= 60:
    print("D")
else:
    print("F")
```
