## ⏰ 하루 1.5시간 (90분) 코딩 루틴

1. **[15분] 웜업:** - `data = list(map(int, input().split()))` 코드를 안 보고 짤 수 있을 때까지 노트에 쓰고 컴퓨터에 5회 타이핑합니다.
2. **[20분] 종이 로직 설계:** - 컴퓨터 화면을 끄고, 줄노트에 문제의 입력 변수와 출력 변수를 적은 뒤 한글로 처리 순서(1, 2, 3...)를 작성합니다.
3. **[40분] 코드 구현 및 디버깅:** - 종이에 적은 로직을 코드로 옮깁니다. `for`문이나 `if`문 내부에는 반드시 `print()`를 넣어 변수 값이 실시간으로 변하는 과정을 눈으로 확인합니다.
4. **[15분] 깃허브 제출:** - 완성 여부와 상관없이 당일 코드를 업로드합니다. 실패한 코드는 상단에 `# 에러 원인`을 주석으로 기록합니다.
